# AI Voice Summarizer

## Project Overview
This project is an AI-powered voice note summarizer. It allows users to upload or record audio, and the application generates concise summaries of the content using advanced AI models.

## Live Demo
Check out the live app here: [AI Voice Summarizer Live](https://aivoicesummarizer.vercel.app/)

## Features
- Upload or record voice notes
- Real-time transcription and summarization
- Easy-to-read summary output
- Modern, interactive UI built with React and Tailwind CSS
- Fully responsive and mobile-friendly

## Technologies Used
- React
- Vite
- TypeScript
- Tailwind CSS
- shadcn-ui
- AI models for voice-to-text and summarization
